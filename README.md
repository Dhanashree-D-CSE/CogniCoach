Cognicoach

Cognicoach is an Android application designed to assist dementia patients and their caregivers by providing essential tools for safety, routine management, and quick access to important information. The app enables users to securely store documents, track patient location in real time, set reminders, and manage daily tasks — all in one accessible and easy-to-use platform.

Technologies

Language: Java 
Framework: Android Studio
Database & Auth: Firebase Authentication, Firebase Realtime Database, Firebase Storage
Other Tools: Google Maps API (for location tracking), Android AlarmManager & Notifications

Features

1.Secure Document Storage
Store medical records, ID proofs, prescriptions, and other important documents in Firebase Storage for quick access during emergencies.

2.Real-Time Location Tracking
Uses Google Maps API to track the patient’s live location, helping caregivers ensure safety and respond quickly in case of wandering.

3.Smart Reminder System
Set reminders for medications, appointments, and daily routines using Android’s notification system to support memory assistance.

4.Daily Task Management
Simple interface to create and manage routine tasks and reminders for better day-to-day organization.

5.Firebase-Powered Authentication
Secure login and data storage using Firebase Authentication and Realtime Database for reliable cloud-synchronized access.
